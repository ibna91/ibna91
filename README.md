## Projects

### [Packtune Preview]
A C# deployment orchestration tool for PSADT packaging, Intune Win32 prep, Windows Sandbox installer analysis, and deployment verification.
Packtune is designed to make Windows application packaging more structured, repeatable, and easier to validate before Intune production deployment.
The application supports the whole flow from an installer.exe -> creted, published and assigned Intune app.
